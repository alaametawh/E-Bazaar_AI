# 🛒 Bazzar – E-Commerce Web Application

Bazzar is a modern and responsive e-commerce web application built with **React**, **TypeScript**, and **Supabase**. It provides a fast, user-friendly shopping experience with secure authentication, product management, and an intuitive admin dashboard.

---

## 🚀 Features

### 👤 Authentication
- Secure user authentication with Supabase
- Login & Logout
- Protected routes
- Session management

### 🛍️ Shopping Experience
- Browse products by category
- Product details
- Responsive product grid
- Search and filtering
- Clean and modern UI

### 🛒 Cart System
- Add products to cart
- Remove products
- Update quantities
- Calculate total price
- Persistent cart state

### 📦 Admin Dashboard
- Dashboard overview
- Manage products
- Create new products
- Edit existing products
- Delete products
- View products count
- Order management

### 📱 Responsive Design
- Mobile-first design
- Tablet support
- Desktop optimized
- Smooth user experience

## ⚡ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/bazzar.git
```

### Navigate to the project

```bash
cd bazzar
```

### Install dependencies

```bash
npm install
```

### Create environment variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_anon_key
```

### Start the development server

```bash
npm run dev
```
